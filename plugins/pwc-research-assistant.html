In brief

  • Research used to eat 2–3 days per engagement: reading the PRD, synthesising it into IA, running workshops to align on personas. This plugin compresses that to minutes.
  • Output is Figma-native — not a PDF report, not a slide deck. It lands on your canvas and is immediately editable.
  • Devil's advocate mode doesn't just generate. It interrogates the PRD and surfaces what's missing before anyone has to ask.

Why I built it

Every new PwC engagement starts the same way: a PRD (or something close to one), a kickoff call, and then days of synthesis work before a designer can put anything meaningful on canvas. Reading, re-reading, extracting personas, sketching journeys — all before the real design thinking begins.

The problem isn't effort. It's that the effort is low-signal. A designer spending three days summarising a 40-page document is not doing design work. They're doing extraction.

So we automated the extraction. Paste the PRD. The plugin reads it, generates the full research layer on canvas — IA, personas, journeys, flows, wireframe starters — and surfaces the gaps the document left open. Designers go straight to the judgment calls.

How it works

  • Ingest: paste your PRD text (or upload a document) directly into the plugin panel.
  • Parse and model: Claude Opus reads the document, extracts user goals, pain points, system boundaries, and feature scope.
  • Generate on canvas: outputs IA diagrams, persona cards, user journey maps, and flow diagrams as native Figma frames — positioned, labelled, and linked.
  • Wireframe starters: for each key flow, generates low-fidelity wireframe screens mapped to the journey steps.
  • Devil's advocate mode: a separate pass where Claude interrogates the PRD — surfacing unstated assumptions, missing edge cases, and contradictions. Displayed as an annotation layer on canvas.

Stack

  • Figma Plugin API — all output is native frames, shapes, and text; nothing is an image export
  • Claude Opus via Anthropic SDK — handles both the synthesis and the adversarial review pass
  • Custom canvas layout engine — positions generated artifacts on a structured grid so output is immediately navigable
