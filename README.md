# Portfolio

Personal portfolio site — single page, vanilla HTML/CSS/JS.

## Stack
- HTML, CSS, JS (no framework, no build step)
- Inter via Google Fonts
- ⌘K command palette + light/dark theme toggle

## Run locally
Just open `index.html` in a browser, or:

```bash
python3 -m http.server 3000
```

Then open `http://localhost:3000`.

## Deploy
Auto-deployed via Vercel on every push to `main`.
